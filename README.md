# Pickly

Pickly는 대량으로 촬영한 사진 속에서  
**고민 없이 좋은 사진을 빠르게 선택할 수 있도록 돕는 앱입니다.

사진을 관리하거나 정리하는 데서 오는 피로를 줄이고,  
넘기면서 선택하고, 선택된 결과만 남기는 경험에 집중합니다.

---

## 문제 정의

아기 사진이나 여행 사진처럼 연사 촬영이 많은 경우  
짧은 시간 안에 수백 장의 사진이 쌓이게 됩니다.

이때 사진을 고르는 과정은
- 비슷한 사진을 반복해서 비교해야 하고
- 선택 기준을 계속 고민해야 하며
- 정리 자체가 부담으로 느껴지기 쉽습니다.

Pickly는 이러한 **사진 선택 과정의 피로를 줄이기 위해** 만들어졌습니다.

---

## 핵심 가치

- **빠른 선택**  
  고민보다 흐름에 집중할 수 있는 인터랙션

- **단순한 구조**  
  필요한 기능만 제공하는 명확한 화면 구성

- **결과 중심**  
  선택이 끝난 사진을 다시 보고 싶게 만드는 경험

---

## 주요 기능

- 특정 폴더의 사진 불러오기
- 풀스크린 뷰어에서 사진을 넘기며 빠르게 확인
- 사진별 채택 / 제외 선택
- 선택 상태에 따른 사진 필터링
- 선택된 사진 일괄 처리
  - 이동/ 복사 / 공유 / 삭제
